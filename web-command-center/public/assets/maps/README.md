# 地图 BP 缩略图

请把每张地图的代表性截图放在这个目录。

推荐规格：
- 尺寸：640 × 360 px
- 比例：16:9
- 格式：jpg / webp / png
- 文件大小：建议 100KB - 300KB，单张不超过 500KB
- 不要带 HUD、准星、聊天框、比分条
- 保留地图最有辨识度的区域，主体尽量在中间 70% 范围内

推荐命名：
- de_dust2.jpg
- de_inferno.jpg
- de_mirage.jpg
- de_nuke.jpg
- de_overpass.jpg
- de_vertigo.jpg
- de_ancient.jpg
- de_anubis.jpg
- de_train.jpg
- de_cache.jpg

网页会按地图名自动寻找这些文件；如果没有图片，会显示占位卡片。
