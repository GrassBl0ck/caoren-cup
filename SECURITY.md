# Security Policy

## Reporting a Vulnerability

If you find a security issue, please do not open a public issue with exploit details or private credentials.

Please contact the maintainer privately, then open a public issue only after the sensitive details have been removed.

## Sensitive Information

Do not commit:

- Admin passwords
- Plugin tokens
- Server IPs or private deployment paths
- `.env` files
- `caoren_config.json`
- `ecosystem.config.cjs`
- Compiled `.dll` / `.pdb` files
- `node_modules/`
- `bin/` or `obj/`
